# Week 1 — Forward Pass Prerequisite Map

## Assignment Goal

Understand and implement the flow:

Word  
→ Vector  
→ Hidden Layer  
→ Raw Scores  
→ Softmax  
→ Prediction

The objective is not to copy the provided implementation.

The objective is to understand every transformation performed during the forward pass.

---

## Learning Rule

Prerequisites are learned in dependency order.

A prerequisite is complete only when I can:

1. Explain it in my own words.
2. Demonstrate it using a small example.
3. Explain where it appears in the assignment.
4. Use it without copying a completed solution.
