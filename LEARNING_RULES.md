# Learning Rules

## Rule 1

Never copy code without understanding it.

---

## Rule 2

Always ask WHY before HOW.

---

## Rule 3

Every concept must answer

- Definition
- Why introduced?
- Problem solved
- Intuition
- Mathematics
- Code
- Advantages
- Limitations
- Real-world usage

---

## Rule 4

Complete the assignment yourself.

Use AI as a mentor, not as a code generator.

---

## Rule 5

Experiment.

After every assignment, change something.

Examples

- Remove a layer
- Change activation
- Increase neurons
- Reduce neurons

Observe.

Learn.

---

## Rule 6

If something is confusing, go deeper instead of skipping it.

---

## Rule 7

Learning is not complete until you can explain it without looking at notes.

---

## Rule 8

Every week ends with

- Reflection
- Cheat Sheet
- Interview Questions
